# 🎤 Claude Code Hands-On Workshop

A 60-minute workshop introducing Claude Code through daily use scenarios.

**Format**: 40 min talk + live demos → 20 min hands-on

**Audience**: Students, teachers, engineers, entrepreneurs — anyone who uses a computer.

## Structure

```
slides/          — Presentation slides
demos/           — Live demo materials
  scenario-1/    — Data wrangling (dirty CSV)
  scenario-2/    — Document automation (meeting notes)
  scenario-3/    — Quick web tool (landing page)
  scenario-4/    — Debug anything (buggy code)
  scenario-5/    — Batch processing (file rename)
  scenario-6/    — Email drafting
  scenario-7/    — Personal automation (multi-agent)
  three-prompts/ — Same problem, three prompt styles
handson/         — Hands-on challenge materials
resources/       — QR codes, install guide, links
```

## Key Message

> "Claude Code isn't just for coders — it's for anyone who uses a computer."

## Quick Start (for attendees)

```bash
# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Start
claude
```
