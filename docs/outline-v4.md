# 🎤 Claude Code Hands-On Workshop — v4（Daily Use 重心版）

**講者**: Yuan
**時長**: 60 min（40 min talk + 20 min hands-on）
**觀眾**: 混合（學生、老師、工程師、企業家）
**語言**: 英文
**日期**: 2026-04-05（六）

---

## Part 1: Talk + Live Demo（40 min, 18 slides）

### Opening（5 min, 3 slides）
1. 舉手互動：「Who spent time on something boring today?」
2. **Shock demo**：觀眾喊一個日常煩事 → Claude Code 60 秒解決
3. 一句定調：「Claude Code isn't just for coders — it's for anyone who uses a computer」

### What is Claude Code（3 min, 2 slides）
4. 一句話：「A smart assistant that lives in your terminal and can actually DO things」
5. 跟 ChatGPT 的差別：ChatGPT 給你答案 → Claude Code 幫你做完

### Daily Use Scenarios — The Core（25 min, 10 slides）

每個 scenario：**30 秒講情境 → 1-2 分鐘 live demo → 一句 takeaway**

**Scenario 1：📊 Data Wrangling（人人都遇到）**
- 情境：「老闆丟了一個亂七八糟的 Excel 要你整理」
- Demo：丟一個髒 CSV → Claude Code 自動清洗、轉格式、產統計
- 🎯 學生：整理實驗數據 / 老師：整理成績 / 企業家：整理客戶名單

**Scenario 2：📝 Document Automation（非工程師最愛）**
- 情境：「把 10 份 meeting notes 整理成一份 summary report」
- Demo：丟幾個 .txt → Claude Code 讀完、分類、產出 markdown 報告
- 🎯 「This is not copy-paste. It reads, understands, and synthesizes.」

**Scenario 3：🌐 Quick Web Tool（學生 + 企業家）**
- 情境：「明天要 demo，需要一個簡單網頁」
- Demo：「Build me a landing page for a bubble tea shop, modern design, with a menu section」
- 🎯 60 秒出一個能看的頁面，觀眾會 wow

**Scenario 4：🐛 Debug Anything（工程師 + 學生）**
- 情境：「這段 code 跑不動，error message 看不懂」
- Demo：貼一段有 bug 的 code → Claude Code 找到問題、修好、解釋原因
- 🎯 「It doesn't just fix — it teaches you why」

**Scenario 5：🔄 Batch Processing（所有人都有的痛）**
- 情境：「200 張照片要改名、100 個 PDF 要合併、rename 一堆檔案」
- Demo：「Rename all files in this folder from IMG_xxxx to 2026-03-30_xxxx, sorted by date」
- 🎯 5 秒搞定手動要花 30 分鐘的事

**Scenario 6：✉️ Email / 文件草稿（企業家 + 老師）**
- 情境：「要寫一封英文回信但不確定語氣對不對」
- Demo：給 context + 對方的信 → Claude Code 幫你起草 + 給你三個版本（formal / friendly / brief）
- 🎯 不是翻譯，是幫你想好怎麼回

**Scenario 7：🤖 Personal Automation（進階 wow moment）**
- 情境：「我想每天早上自動收集 AI 新聞 + 整理成摘要」
- Demo：秀你的真實系統 —— 你的 6 agent 公會，修塔爾克每天自動做的事
- 🎯 「This is what "daily use" looks like when you go all in」

### The Secret Sauce（5 min, 2 slides）
15. 「同題三解」快閃版：同一個需求，lazy prompt vs structured prompt → 品質差超多
16. Tips：CLAUDE.md = 你的 AI 的 SOP、MCP = 給 AI 裝外掛

### Close（2 min, 1 slide）
17. 三大坑一句話帶過：token 爆 / 幻覺 / 權限
18. QR code → 安裝指引 + 資源包 → 「Now it's your turn」

## Part 2: Hands-On（20 min）

### Challenge 1（8 min）— Pick Your Scenario
觀眾從清單中選一個最有感的場景自己做：
- A. 整理一份 CSV 資料
- B. 做一個小網頁
- C. 修一段有 bug 的 code
- D. 批次處理檔案
（你巡場協助）

### Challenge 2（10 min）— Your Own Problem
「用 Claude Code 解決你今天真正遇到的一個問題」
- 完全自由題
- 你巡場，挑有趣的作品大螢幕展示

### Showcase + Close（2 min）
- 秀 2-3 個觀眾作品
- 「The best way to learn AI is to build with it — and you just did.」
